# Whack-a-Mole Reaction Robot

An Arduino-based reaction game robot inspired by the classic Whack-a-Mole game.

## Features

* Fast reaction gameplay
* LCD score display
* Increasing difficulty levels
* Buzzer sound effects
* High score system

## Components Used

* Arduino UNO
* LCD Display
* Push Buttons
* LEDs
* Buzzer
* Jumper Wires

## How It Works

Players must quickly press the correct button when the LED lights up. The game speed increases as the player progresses through levels.
